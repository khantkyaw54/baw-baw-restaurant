# BAW BAW 居酒屋（Izakaya）

このリポジトリは、**ミャンマー居酒屋「BAW BAW」**の公式Webサイトのソースコードをまとめたものです。  
名古屋にある実店舗を想定し、メニュー紹介・店舗情報・お知らせ・お客様の声などを掲載しています。

---

## 📌 サイト概要

- 店舗名：BAW BAW（バウバウ）
- ジャンル：ミャンマー家庭料理・居酒屋
- 対象：日本のお客様・初来店の方
- 目的：店舗紹介・集客・情報発信

---

## ✨ 主な機能・特徴

- 📱 **レスポンシブ対応**（スマートフォン / PC）
- 🍽 メニュー紹介ページ
- 🗺 Google Maps 埋め込み
- 💬 お客様の声（レビュー表示）
- 🍔 ハンバーガーメニュー（モバイル対応）
- 🎨 SCSS を使用したスタイリング
- ⚡ 画像最適化（WebP形式）
- ⚛️ 一部セクションで React を使用

## 🚀 ローカルでの表示方法
